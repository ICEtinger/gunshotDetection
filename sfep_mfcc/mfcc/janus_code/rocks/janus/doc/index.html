This page serves as an entry point to the different parts of the JRTk documentation available. Currently, the documentation to the Janus Recognition Toolkit consists of the following parts:

  • Online documentation for different parts of the system (might in parts be out-of-date, accessible from inside ISL only)
  • The comprehensive "JRTk and Ibis Manual" is available in different formats:
    • Portable Document Format (pdf)
    • PostScript Book (ps)
    • HTML-Pages
    The documentation in this directory can be rebuilt from LaTeX sources using latex, pdftex, and latex2html, please have a look at the Makefile.

The manual will eventually contain the most up-to-date information from all the above documentation sources, which -- you have been warned -- might in parts be out-of-date.

Please note that there is online help available for all Janus modules and methods by typing "module -help" or "module method -help" at the Tcl prompt. There is also documentation available for the Language Modelling Toolkit and the ISL example system, which are distributed separately.

If you're looking for example data and systems, check the ISL internal web-pages.
