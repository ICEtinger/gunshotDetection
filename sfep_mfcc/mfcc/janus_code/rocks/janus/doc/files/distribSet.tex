\section{\Jlabel{file}{distribSet}}

The description file used in a \Jref{module}{DistribSet}. An example
looks like this:

\begin{verbatim}
; -------------------------------------------------------
;  Name            : distribSetISLci
;  Type            : DistribSet
;  Number of Items : 199
;  Date            : Thu Jul 11 20:21:13 2002
; -------------------------------------------------------
+QK-b            +QK-b
+QK-m            +QK-m
+QK-e            +QK-e
SCH-b            SCH-b
SCH-m            SCH-m
SCH-e            SCH-e
SIL-m            SIL-m
T-b              T-b
T-m              T-m
T-e              T-e
\end{verbatim}

The second column tells you which codebook to use.
